# How to Get Your Twilio Sandbox Join Code

## Method 1: From Twilio Console (Easiest)

1. Go to: https://console.twilio.com/us1/develop/sms/try-it-out/whatsapp-learn
2. You'll see a section called **"Sandbox Configuration"**
3. Look for text that says:
   ```
   To connect your sandbox, send "join <word>-<word>" to +1 415 523 8886
   ```
4. Your join code will be something like:
   - `join happy-tiger`
   - `join blue-mountain`
   - `join red-ocean`

## Method 2: Direct Link

Visit this URL (you must be logged in):
https://console.twilio.com/us1/develop/sms/try-it-out/whatsapp-learn

## Method 3: From Main Console

1. Go to: https://console.twilio.com/
2. Click **"Messaging"** in left sidebar
3. Click **"Try it out"**
4. Click **"Send a WhatsApp message"**
5. You'll see the join code on that page

## What to Do Once You Have It:

1. Open WhatsApp on your phone
2. Start a new chat with: **+1 415 523 8886**
3. Send message: `join your-code-here` (e.g., `join happy-tiger`)
4. Wait for confirmation message from Twilio
5. You're ready to receive OTP messages!

## Example:

If your code is `happy-tiger`, send:
```
join happy-tiger
```

You'll get a reply like:
```
Joined happy-tiger! You can now send and receive messages.
```
